# 📚 Study Planner App

An Android Study Planner application built using Java and Firebase.

## 🚀 Features
- Phone OTP Login using Firebase
- Add Subjects
- Add Study Tasks
- Set Deadlines
- Firebase Firestore Database
- Modern UI Design
- RecyclerView Integration

## 🛠 Technologies Used
- Java
- Android Studio
- Firebase Authentication
- Cloud Firestore
- RecyclerView
- Material Design

## 📱 Screens
- Login Screen
- OTP Verification
- Dashboard
- Add Subject
- Task Management
